## Installing
Make sure you have the following installed locally in your system:

* Go 1.26.0
* Docker

### Running backend
```bash
git clone https://github.com/jacomemateo/OutaStock/
cd OutaStock
make backend # This will start the local PostgreSQL server in a docker container
```

### Running Frontend
```
npm install
npm run dev
```
